# 📝 To-Do List Manager

A simple command-line **To-Do List Manager** built using Python. This project allows users to manage daily tasks through an interactive menu-driven interface. Users can add new tasks, view all saved tasks, mark tasks as completed, and delete tasks. Task data is stored in a JSON file, allowing it to persist even after the program is closed.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Save tasks automatically using JSON
- 🔄 Load previously saved tasks on startup
- 📌 Automatic task ID generation
